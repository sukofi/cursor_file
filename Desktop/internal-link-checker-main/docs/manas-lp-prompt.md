# Manas用 LP作成プロンプト

以下をManasにそのまま貼り付けて、このSaaSのランディングページを生成してください。

---

## プロンプト本文

### 依頼
「内部リンクマトリクス」というSaaSのランディングページ（LP）を作成してください。見込み客が「無料で始める」または「料金を見る」に進みたくなるよう、訴求力のある1ページ構成でお願いします。

### 製品概要
- **製品名**: 内部リンクマトリクス（Internal Link Matrix）
- **一言**: ブログ・メディアの記事間の内部リンクを、スプレッドシートと連携してマトリクス形式で可視化するWebツールです。
- **主な流れ**: Googleスプレッドシート（KW・URL一覧）と連携 → 各URLをクロールして内部リンクを抽出 → どの記事がどの記事にリンクしているかをマトリクスで一覧表示 → CSVエクスポート可能。

### ターゲット
- ブログ・オウンドメディアを運営している担当者
- SEOや内部リンク設計を意識しているコンテンツマーケター・編集者
- 複数サイトや複数プロジェクトの内部リンクを管理したい人
- スプレッドシートで記事一覧をすでに管理しているチーム

### 訴求したい価値・ベネフィット
1. **可視化**: 記事同士のリンク関係が一覧で分かるので、リンク切れ・リンク過多・リンク不足の記事をすぐ把握できる。
2. **スプレッドシート連携**: 既存の記事一覧（KW・URL）をそのまま使え、設定はスプレッドシートIDと列指定だけ。
3. **柔軟な計測**: CSSセレクタで「このブロック内のリンクだけ数える」など、内部リンクの定義を自分で設定できる。
4. **複数プロジェクト**: サイトごとにプロジェクトを作成し、ダッシュボードから切り替えて管理できる。
5. **手軽さ**: 無料プランで試せ、本格運用時は月額980円のProプランで行数無制限。

### 料金（記載する内容）
- **Free**: ¥0/永久無料 — スプレッドシート10行まで、プロジェクト無制限、マトリクス表示・CSVエクスポート
- **Pro**: ¥980/月 — スプレッドシート行数無制限、上記に加えて優先サポート、いつでもキャンセル可能

### LPに含めてほしいセクション（順番の目安）
1. **ヒーロー**: キャッチコピー + 短い説明 + CTA（「無料で始める」「料金を見る」）
2. **課題・共感**: 「記事が増えると、どの記事にどこからリンクしているか分からなくなる」「内部リンクの偏りや孤立記事を手作業で確認するのは大変」といった悩みに触れる
3. **解決・仕組み**: スプレッドシート連携 → クロール → マトリクス表示の流れを、図や短いステップで説明
4. **機能・特徴**: 上記の価値（可視化・スプレッドシート連携・CSSセレクタ・複数プロジェクト・無料で試せる）を箇条書きまたはアイコン付きで
5. **料金**: Free / Pro の2プランを簡潔に。Proは「人気」バッジを付ける想定
6. **FAQ**: 「いつでもキャンセルできるか」「支払い方法」「無料から有料への移行」など1〜3問
7. **最終CTA**: 再度「無料で始める」ボタンと、ログイン/新規登録への導線

### トーン・スタイル
- 専門用語は必要最小限にし、ブログ・メディア運営者に伝わる平易な日本語
- 押し売りせず、「まず無料で試せる」安心感を出す
- 信頼感のある落ち着いたトーン（SEO・コンテンツ担当者が仕事で使うツールとしてふさわしい印象）

### 技術・導線の前提
- ヘッダーに「ログイン」リンク、CTAは「無料で始める」でサインアップ/ログイン画面へ、「料金を見る」で料金ページへ飛ぶ想定
- 製品名は「内部リンクマトリクス」で統一
- 料金は「Free」「Pro」「¥0」「¥980/月」の表記で統一

### デザインの希望（任意）
- 清潔で読みやすいレイアウト。ダッシュボードやマトリクス画面のスクリーンショット/モックがあれば hero や機能説明に使うと説得力が増す
- CTAボタンは目立つ色で1〜2種類に統一
- モバイルでも読みやすい1カラム〜2カラム中心

---

以上を元に、このSaaSのLP用のコピーと構成案（見出し・各セクションの文面・CTA文言）を出力してください。HTML/Reactのコードまで出す必要はなく、コピーと構成で大丈夫です。

## 短縮版プロンプト（文字数制限がある場合）

「内部リンクマトリクス」のLPを作成してください。

**製品**: ブログ・メディアの記事一覧（Googleスプレッドシート）と連携し、記事間の内部リンクをマトリクスで可視化するSaaS。クロールでリンクを抽出し、どの記事がどこにリンクしているか一覧表示・CSV出力できる。

**ターゲット**: ブログ・メディア運営者、SEO・コンテンツ担当者。

**訴求**: 内部リンクの可視化、スプレッドシート連携、CSSセレクタで計測範囲を指定可能、複数プロジェクト対応。無料プラン（10行まで）で試せ、Proは月額980円で行数無制限。

**LP構成**: ヒーロー（キャッチ+CTA）→ 課題共感 → 仕組み説明 → 機能・特徴 → 料金（Free/Pro）→ FAQ → 最終CTA。トーンは平易で信頼感のある日本語。CTAは「無料で始める」「料金を見る」。
