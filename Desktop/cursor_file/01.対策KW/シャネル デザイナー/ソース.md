## 4. 選定理由の分析：なぜブレイジーを選んだのか

### 4-1. 内部昇格 vs 外部招聘

#### 歴史的背景
- **115年間で4人のデザイナー**：ココ→ラガーフェルド→ヴィアール→ブレイジー  
- **伝統的パターン**：ヴィアールまでは内部昇格（30年以上在籍後）  
- **今回の転換**：外部招聘へ方針転換  
  出典：[Vogue Japan](https://www.vogue.co.jp/article/chanel-appoints-matthieu-blazy-as-artistic-director)

#### 外部招聘の理由
- **新風導入**：内部昇格では限界、マンネリ打破  
- **クラフトマンシップと大胆さの両立**：ボッテガでの実績が評価  
- **若返り戦略**：40歳の新世代デザイナー起用  
  出典：[FashionSnap](https://www.fashionsnap.com/article/chanel-matthieu-blazy/)、[Affectus Design](https://affectusdesign.com/2024/12/22/modetext-chanel-matthieublazy/)

### 4-2. 他候補との比較

#### 有力候補として噂された人物
- **エディ・スリマン**（Hedi Slimane）：セリーヌ離脱後の有力候補  
- **サイモン・ポート・ジャックムス**（Simon Porte Jacquemus）：若手注目株  
- **フィービー・ファイロ**（Phoebe Philo）：セリーヌ時代の功績  
- **ジョン・ガリアーノ**（John Galliano）：メゾン マルジェラでの復活劇  
  出典：[Vogue Japan](https://www.vogue.co.jp/article/chanel-appoints-matthieu-blazy-as-artistic-director)、[WWD JAPAN](https://www.wwdjapan.com/articles/2089078)、[Highsnobiety](https://highsnobiety.jp/p/matthieu-blazy-bottega-veneta-chanel-louise-trotter/)

#### ブレイジーが選ばれた理由
- **クラフトマンシップと現代性のバランス**：ボッテガでの手仕事重視姿勢がシャネルのDNAと合致  
- **アトリエとの対話重視**：チームワーク型デザイン開発  
- **「実用的なラグジュアリー」哲学**：シャネルの「女性の解放」理念と親和性  
  出典：[Vogue Japan（ブレイジーインタビュー）](https://www.vogue.co.jp/article/matthieu-blazy-chanel-december-2025)、[Ameblo](https://ameblo.jp/mika0405-2021/entry-12939664779.html)
