# 不確実性認識型コンフォーマル予測と世界モデル強化学習による安全な都市交通制御

## 原題
Safe Urban Traffic Control via Uncertainty-Aware Conformal Prediction and World-Model Reinforcement Learning

## アブストラクト
都市交通管理は、将来の状況を予測し、異常を検出し、安全な是正措置を講じると同時に信頼性保証を提供するシステムを必要とします。本研究では、STREAM-RLという統一フレームワークを提案し、以下の三つの新規アルゴリズム的貢献を導入します：(1) PU-GAT+は、不確実性に基づきグラフ注意機構を動的に再重み付けする不確実性誘導適応コンフォーマル予測器であり、信頼度単調注意機構を用いて分布に依存しないカバレッジ保証を実現します；(2) CRFN-BYは、任意の依存関係下でBenjamini-Yekutieliの偽発見率制御を行う正規化フローを用いた不確実性正規化残差のモデル化を行うコンフォーマル残差フローネットワークです；(3) LyCon-WRL+は、リャプノフ安定性証明、認定リプシッツ境界、不確実性伝播を伴う想像ロールアウトを備えた不確実性誘導安全世界モデル強化学習エージェントです。これらにより、予測から異常検出、安全な方策学習まで較正された不確実性を伝播し、エンドツーエンドの理論的保証を持つ初のフレームワークとなります。複数の実世界交通軌跡データにおける実験では、STREAM-RLは91.4％のカバレッジ効率を達成し、検証済み依存関係下で偽発見率を4.1％に制御し、標準的なPPOの69％に対して安全率を95.2％に向上させつつ高い報酬を獲得し、エンドツーエンド推論遅延は23msでした。
