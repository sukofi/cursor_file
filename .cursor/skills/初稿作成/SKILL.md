# 初稿作成・参照・流れ

買取系SEO記事の**初稿**（初稿.md・初稿.html）を作成・編集するときに参照するファイルと、見出し・内部リンクの扱い。

## 参照ファイル一覧

| 場所 | 用途 |
|------|------|
| `01.対策KW/[KWフォルダ]/[KW名] - ④構成.md` | 見出しの出どころ。初稿の h1/h2/h3 はここに準拠する |
| `01.対策KW/[KWフォルダ]/内部リンク候補.md` | 内部リンク候補（KW・URL・意図）。執筆・編集前に読み込む |
| `02.レギュレーション/レギュレーション.md` | リード・表現・まとめ・画像・装飾のルール |
| `02.レギュレーション/大吉訴求文.md` | 訴求ブロックの本文・表・CTA |
| `02.レギュレーション/記事編集ワークフロー-SKILL.md` | 内部リンク形式・語尾チェック・訴求の一元化・HTML/MD同期 |

## 初稿の見出し

- 初稿の**見出し**は、**④構成.md の見出し構成にそのまま準拠**する。
- h1（タイトル）・h2・h3 の文言・順序を ④構成.md から踏襲し、本文執筆時に崩さない。

## 内部リンク候補.md を読み込む流れ

1. **事前に読み込む**：初稿の執筆または編集を始める前に、同一KWフォルダの **`内部リンク候補.md`** を読み込む。
2. **内容**：候補には KW テキスト・URL・意図（なぜそのリンクか）が含まれる想定。執筆・編集時に参照する。
3. **配置**：候補を**1か所にまとめず**、記事の流れ・各見出しの文脈に合わせて**分散配置**する（例：相場の説明の直後→相場系KW、売り方の直後→買取系KW）。
4. 形式・装飾は `02.レギュレーション/記事編集ワークフロー-SKILL.md` の「内部リンク」に従う（【関連記事】＋赤字リンク等）。

## 他スキルとの役割分担

- **構成参照・ルール**：構成作成時の参照ファイル・見出しルール・④構成.md フォーマット。
- **構成作成**：④構成.md の作成・見出しの組み立てと出力。
- **初稿作成**（本スキル）：④構成に沿った見出し＋内部リンク候補の読み込み流れ。編集時の細則は記事編集ワークフローへ。
