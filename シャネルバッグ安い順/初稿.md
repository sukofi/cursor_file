## **方法2：本場フランスの正規店で買う｜デタックスで最大12〜13%お得に**

旅行の機会があれば、本場フランスの正規店で購入するのがお得です。フランスの付加価値税(VAT)は20%ですが※18、旅行者はデタックス手続きを行うことで購入金額の約12〜13%程度の還付を受けることができます※19。

為替レートにも左右されますが、日本での販売価格よりも数万円から十数万円安くなるケースが多いです。

| **3** | 出国時に空港税関で免税書類に承認印を取得 |

| 購入品合計が**20万円以内** | 免税（課税なし）※21 |
| 購入品合計が**20万円超** | 20万円以内の品物は免税、残りは課税※21 |
| **1品だけで20万円超** | **全額が課税対象**※21 |
| バッグ・革製品の関税率 | 関税+消費税で約15〜20%程度※22 |

**その他の注意事項：**

* 為替レートにより総額は変動するため、購入時のユーロ/円レートを確認してください  
* シャネルは一部店舗で購入制限（1人1点まで等）を設けている場合があります

## **方法3：並行輸入とポイント還元を併用｜楽天で実質10%オフも可能**

楽天市場などの大手ECサイトに出店している並行輸入店を利用する方法もあります。並行輸入品は、海外の正規代理店から直接買い付けた商品であるため、国内定価よりも安く設定されていることがあります。

### **並行輸入のメリット・デメリット**

| メリット | デメリット |
| ----- | ----- |
| ✓ 国内定価より安い場合が多い | ✗ 国内正規店のアフターサービスに制限 |
| ✓ ポイント還元で実質10%以上OFF可能 | ✗ 在庫が限定的 |
| ✓ 自宅で完結、渡航不要 | ✗ 真贋判断が必要 |

さらに「お買い物マラソン」などのポイントアップキャンペーンを併用すれば、実質10%以上の還元を受けることも可能です。ただし、国内正規店でのアフターサービスが一部制限される可能性がある点は留意しておきましょう。

## **中古シャネルで失敗しない選び方｜資産価値が高いモデルと偽物の見分け方**

中古でシャネルを購入する際、単に「安いから」という理由だけで選ぶのは非常に危険です。将来的な売却価格を見据えた「資産価値」と、偽物を掴まないための「防衛策」を知っておきましょう。

## **資産価値が高い「色・素材・モデル」の共通点**

シャネルの中でも、将来的に高く売れる(リセールが良い)条件は明確です。

### **【資産価値が高い】シャネルの条件ランキング**

| 順位 | 条件 | リセール率 | 理由 |
| ----- | ----- | ----- | ----- |
| **1位** | **黒×キャビアスキン×ゴールド金具** | 80〜90% | 最も需要が高く、傷が目立ちにくい |
| 2位 | ベージュ×キャビアスキン | 75〜85% | 定番色で年齢問わず人気 |
| 3位 | 黒×ラムスキン | 70〜80% | 高級感があるが傷つきやすい |
| 4位 | その他定番色 | 60〜70% | ネイビー、グレーなど |
| 5位 | 限定色・トレンド色 | 40〜60% | 流行に左右され値崩れしやすい |

**最も価値が落ちにくいのは「黒×キャビアスキン×ゴールド金具」の組み合わせです。**

キャビアスキンは牛革に型押しを施した素材で、傷が目立ちにくく耐久性に優れているため、中古市場での需要が絶えません。また、流行色のバッグは値崩れしやすいですが、黒やベージュの定番色は時代を問わず高値で取引されます。

## **偽物を避ける！信頼できるショップの見極め方**

偽物を避けるための最大のポイントは、2021年以降の仕様変更を理解することです。

### **シャネルの真贋システム変遷**

| 時期 | 真贋システム | 特徴 |
| ----- | ----- | ----- |
| **〜2021年春** | シリアルシール+ギャランティカード | 7〜8桁の番号が記載※26 |
| **2021年春夏〜** | **RFID（マイクロチップ）**※23 | 8桁の固有番号付き金属プレート※24 |

以前のモデルには「シリアルシール」と「ギャランティカード」が付属していましたが、2021年春夏コレクション以降の新作からは「マイクロチップ(RFID)」を内蔵した金属プレートに移行しています※23。

「付属品が揃っているから本物」と思い込まず、最新の鑑定技術を持つショップで購入することが不可欠です。また、不自然に価格が安すぎる個人売買やフリマアプリでの購入は、リスクが非常に高いため推奨されません。

### **RFIDタグ（マイクロチップ）の詳細**

| 項目 | 詳細 |
| ----- | ----- |
| **導入時期** | 2021年春夏コレクションから※23 |
| **外観** | シャネルのロゴマークの下に8桁の固有番号が印字された金属プレート※24 |
| **読み取り** | 内部にICチップが埋め込まれており、専用機器で読み取り可能 |
| **重要** | 一般の携帯アプリでは読み取れないため、専門店での鑑定が必須 |

### **2021年以前のモデル（シリアルシール）の真贋ポイント**

| チェック項目 | 本物の特徴 | 偽物の特徴 |
| ----- | ----- | ----- |
| **シールの剥がれやすさ** | 簡単に剥がれない※25 | 簡単に剥がれる |
| **数字の印字** | 7〜8桁の数字が明瞭に印字※26 | 印字が粗い、不明瞭 |
| **番号の一致** | ギャランティカードとシール番号が一致 | 番号が不一致 |

### **信頼できる買取店の条件チェックリスト**

| ✓ | 条件 | 確認方法 |
| ----- | ----- | ----- |
| □ | 古物商許可証の掲示がある | 店舗内に許可証番号が掲示されているか |
| □ | 真贋鑑定の専門部署・鑑定士が在籍 | 公式サイトで鑑定体制を確認 |
| □ | 偽物だった場合の返金保証制度がある | 購入前に保証内容を確認 |
| □ | 日本流通自主管理協会(AACD)加盟店である※27 | AACD公式サイトで加盟店を確認 |

## **よくある質問(FAQ)**

### **Q：シャネルが一番安く買える時期はいつ?価格改定の傾向は?**

シャネルは年数回のペースで世界的に価格改定(値上げ)を行っています。そのため「今日が一番安い日」と言われるほど、早めの購入が推奨されます。

### **シャネル価格改定の傾向**

| 年 | 改定回数 | 傾向 |
| ----- | ----- | ----- |
| 2021〜2024年 | 年4回※13 | 継続的な値上げ |
| 2025年 | 3月・5月・8月に実施済 | 今後も継続予定 |

傾向として、原材料費の高騰や為替の変動、ブランド価値の維持を理由に、一度上がった定価が下がることは極めて稀です。購入を迷っている間にも値上げが発表される可能性があるため、欲しいモデルが決まったら早めに決断するのが賢明です。

### **Q：中古で買うのは恥ずかしい?新品との違いは?**

近年はサステナビリティ(持続可能性)への意識の高まりや、ヴィンテージファッションの流行により、中古ブランド品を持つことは一般的であり、恥ずかしいことではありません。

### **中古品購入の社会的評価**

| 側面 | 現代の評価 |
| ----- | ----- |
| サステナビリティ | ✓ 環境に配慮した選択として評価される |
| ファッション性 | ✓ ヴィンテージは「ファッション感度が高い」証 |
| 経済性 | ✓ 賢い買い物として肯定的に捉えられる |

むしろ、現行品にはない独特の風合いを楽しめるヴィンテージシャネルは、ファッション感度の高い層から絶大な支持を得ています。新品の完璧な美しさを選ぶか、中古で賢く憧れを手にするかは、個人のライフスタイル次第です。

### **Q：古いシャネルバッグは修理できる?リペア可能な範囲は?**

シャネルはアフターサービスが充実しており、古いヴィンテージ品であっても正規店(シャネル・リペアセンター)で修理を受け付けてくれます。

### **シャネル正規リペアの対応範囲**

| 修理内容 | 対応可否 | 参考費用 |
| ----- | ----- | ----- |
| チェーンの革紐交換 | ○ 可能 | 数万円〜 |
| 金具の修理・交換 | ○ 可能 | 数万円〜 |
| 内装の張り替え | ○ 可能 | 数万円〜 |
| 破損が激しい場合 | △ 要相談 | 見積もり必要 |

チェーンの革紐交換や金具の修理、内装の張り替えなど、多くのトラブルはリペア可能です。

ただし、修理代金は数万円単位でかかることが多いため、中古で購入する際は「修理が必要な状態か」を事前によく確認してください。
